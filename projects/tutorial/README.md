## Using the IDE

You can also run these lessons in the Steps IDE:

```bash
python -m steps_ide.main
```

Then navigate to the lesson folder and press **F5** to run!

## Learning Path

1. Start with Lesson 1 and work through sequentially
2. Read the tutorial document (`/docs/TUTORIAL.md`) alongside the code
3. Run each program and observe the output
4. Try the exercises in the tutorial
5. Modify the programs to experiment
6. Build your own projects using what you've learned!
